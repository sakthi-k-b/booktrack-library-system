BookTrack

Efficient Library Management Made Simple.

BookTrack is a full-stack demo project built with Flask and SQLite, providing secure, modern, and intuitive solutions for libraries and users.

Key Features

  • • Secure user registration and login with password hashing
  • • Role-based access: admin & regular users
  • • Comprehensive book management: add, edit, delete, search books
  • • Borrow tracking: assign, return, and filter borrowed books by due status
  • • User management: view, search, and remove users (admin only)
  • • Responsive, modern UI with dark mode support

Technical Details

  • Built with Flask (Python web framework)
  • Data stored in SQLite with foreign key support
  • Password hashing using SHA-256 via hashlib
  • Environment variables via dotenv for secure configuration
  • UI styled with Tailwind CSS for modern theming and dark mode
  • Dynamic interactivity via minimal JavaScript
  • Jinja2 templating for rendering server-side HTML
  • Project intended for technical demo & portfolio
